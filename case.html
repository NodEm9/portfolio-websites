Case Study for myFlix Full-Stack Project

Overview

myFlix is a MERN Stack application designed for movie enthusiasts to explore detailed information on a wide range of movies, directors, and genres. Users can register, log in securely, and manage a personalized list of favorite movies. The app includes robust user authentication and authorization features to ensure a secure and personalized experience.

Purpose & Context

myFlix is a personal project I built as part of my web development course at CareerFoundry to demonstrate my ability to develop a full-stack JavaScript application.

Objective

The goal of the project was to develop a full-stack application that lets users explore movie details, directors, and genres, to showcase my skills in backend and frontend integration as well as database management.

Approach

Server-Side

I developed a RESTful API with Node.js and Express to interface with a MongoDB database, supporting CRUD operations via standard HTTP methods like GET and POST. The API delivers movie information in JSON format, enabling seamless data retrieval and storage.

View endpoints

Testing

I tested the API using Postman and implemented user authentication and authorization with basic HTTP and JWT authentication.

Client-Side

I created a responsive client-side application using React. The application allows users to sign up, log in, and view information about different movies, directors, and genres. Users can also add movies to their list of favorites and edit their profile details.

Chanllenges

This project presented challenges in integrating frontend and backend components to enable seamless communication and data display. I mastered Redux for managing state effectively, implemented secure user authentication and authorization, and ensured responsive design for accessibility across devices. The guidance from my tutor was invaluable in achieving these goals.

Duration

The backend development took about one to two weeks, while the frontend required three weeks due to the additional time needed to master Redux for effective state management and to test the application across various devices.
